系统架构总览

从"谁调谁"的角度看整体部署架构

请求流向
React 管理后台（浏览器）  → 调 /admin/**  →  Spring Boot 后端（Docker 容器）
微信小程序（独立项目）  → 调 /miniapp/**  →  同一个 Spring Boot

后端连接： MySQL 8.0  |  Redis 7.x  |  阿里云 OSS
后端包结构
com.chengjialiye
  admin/  controller / service / service.impl / mapper / dto
  miniapp/  controller / service / service.impl / mapper / dto
  common/  entity / enums / util / config / exception / result
认证流程（轻量）
请求 → TokenInterceptor 取 Header Token → Redis 校验（存在/未过期） → 通过则写入 ThreadLocal → 不通过则返回 401

技术栈确认

后端Java 21 · Spring Boot 3 · MyBatis-Plus · MySQL 8.0 · Redis · OSS · Maven · Docker
前端React 18 · TypeScript · Vite · shadcn/ui · Radix UI · Tailwind CSS · react-router-dom v6 · Zustand · axios · ECharts
认证轻量 Token 校验（拦截器 + Redis，非 Spring Security）
部署Docker，无 MQ
