关键点

  • 管理后台和小程序共用同一个后端服务，通过 URL 前缀区分：/admin/** 和 /miniapp/**
  • 认证拦截器按前缀分别校验：admin 用管理后台 Token，miniapp 用微信授权 Token
  • 小程序前端是独立项目，但调的是同一个后端
  • Docker 部署，一个容器搞定整个后端
  • Redis 存 Token、验证码、配置缓存
  • 阿里云 OSS 存图片/文件、Excel 导出

认证流程（简版）

请求 → TokenInterceptor（从 Header 取 Token） → Redis 校验（是否存在，是否过期） → 通过：写入 ThreadLocal 当前用户 → 不通过：返回 401
