## 8. 前端字段一致性核查

核查依据：`docs/需求文档/需求文档-正式版/07-推广裂变与邀请奖励/管理后台/页面规格/*`。

结论：当前前端页面展示字段与正式版页面规格**未完全一致**。已修复菜单树与关键页面标题，但多数页面仍是简化版列表/详情，且部分规格字段后端 VO 暂未提供，不能只靠前端补齐。

| 页面规格               | 当前一致性 | 主要缺口                                                                                                                                                                             |
| ---------------------- | ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 推广规则配置页         | 不一致     | 规格要求四个可编辑 Tab：普通用户奖励 5 类事件、代理奖励规则组、关系有效期、风控参数；当前仅展示四个 Tab 外壳，主体仍是旧规则列表 + 新增/编辑规则弹层，缺保存二次确认和正式表单字段。 |
| 普通邀请关系列表页     | 不一致     | 缺邀请关系编号精确搜索、邀请人/被邀请人 UUID/手机号列、当前已发放奖励列、绑定时间范围、筛选标签、导出、查看风控原因、标记无效/解除冻结行操作。                                       |
| 邀请关系详情页         | 不一致     | 缺邀请人/被邀请人完整信息分区、首次点击/注册/首次会员/首次充值时间、奖励触发记录、风控命中记录、操作日志、补充备注；当前只展示部分基础字段和时间线。                                 |
| 普通邀请奖励流水页     | 不一致     | 缺奖励流水号/邀请人/被邀请人搜索、时间范围、创建时间、到账时间、跳邀请关系详情、导出；当前与冻结处理共用组件，列里出现审核/冻结处理相关操作。                                        |
| 冻结奖励处理页         | 部分一致   | 已有独立路由/标题/冻结筛选/确认发放/确认无效按钮；缺奖励流水号与用户 UUID/手机号搜索、冻结时间范围、补充备注、并发处理提示、完整冻结原因筛选。                                       |
| 代理列表页             | 不一致     | 缺代理编号、累计扫码/点击数、累计注册数、累计成功邀请人数、累计应发/已发/待结算奖金、奖金规则组必填、终止合作、导出；当前表格为代理/联系人/学校/状态/创建时间。                      |
| 代理详情页             | 不一致     | 规格要求代理信息概览 + 推广明细/奖金明细/结算记录三个 Tab；当前为基础信息 + 统计卡 + 素材 + 最近事件，缺奖金明细、结算记录、导出、被推广用户 UUID/手机号等字段。                     |
| 代理结算管理页         | 不一致     | 缺结算单号/代理编号搜索、结算周期筛选、统计口径说明列、备注列、收款信息占位、导出结算列表/明细；当前只覆盖查询、确认、发放和无手动生成入口。                                         |
| 推广素材与二维码管理页 | 不一致     | 缺代理编号/代理名称/二维码编号搜索、代理编号/所属代理列、二维码素材模板、有效期“永久有效”、下载、查看历史版本、预览弹层、重生成/停用二次确认文案。                                   |

需要后端补字段的典型项：

- 用户 UUID、手机号、脱敏展示权限：邀请关系/奖励流水/代理详情多个页面均需要。
- 奖励触发记录、风控命中记录、操作日志：邀请关系详情需要结构化列表。
- 代理奖金明细、结算记录、二维码历史版本：代理详情和素材页需要对应接口/VO。
- 代理列表统计列：部分可来自 `PromotionAgentStatVO`，但当前页面未展示完整口径。
