## 6. 已验证的关键业务规则

| 规则 | 验证方式 | 结果 |
|------|----------|------|
| H5 内容必须有 URL | 创建时 contentType=H5, contentUrl 为空 | 被拒绝 ✅ |
| NATIVE 内容必须有正文 | 创建时 contentType=NATIVE, contentBody 为空 | 被拒绝 ✅ |
| URL 类型配置值校验 | 批量保存 configType=URL, value="not-a-url" | 被拒绝 ✅ |
| JSON 类型配置值校验 | 批量保存 configType=JSON, value="{invalid" | 被拒绝 ✅ |
| 入口 pageCode+entryKey 唯一 | 重复创建相同组合 | 被拒绝 ✅ |
| H5 入口必须有跳转目标 | jumpType=H5, jumpTarget 为空 | 被拒绝 ✅ |
| 热词 scene+word 唯一 | 重复创建相同组合 | 被拒绝 ✅ |
| 屏蔽词 blockType+word 唯一 | 重复创建相同组合 | 被拒绝 ✅ |
| 未登录访问后台接口 | 无 Token 请求 /admin/** | 返回 401 ✅ |
| 已下线内容不出现在小程序 | 下线后查公告列表 | 不包含 ✅ |
| 已禁用入口不返回给小程序 | 禁用后查入口列表 | 不包含 ✅ |
| 已禁用热词不返回给小程序 | 禁用后查热词列表 | 不包含 ✅ |
| public_visible=0 不暴露 | 设为非公开后查小程序配置 | 不返回 ✅ |
| 操作审计落库 | 文章状态变更/配置保存/屏蔽词操作后查日志 | 有记录 ✅ |
