## 6. 边界（本期不做）

> 这一节专门消灭"首版是否做 X"。看得到但不做的，写明处理方式。

| 不做的能力 | 原因 | 本期处理方式 | 后续计划 |
|------------|------|--------------|----------|
| | | 隐藏 / 置灰 / 空态 / 仅预留数据结构 | 如：V1.1 |

> **处理方式选择指南：**
> - **隐藏**：功能入口完全不展示，用户感知不到
> - **置灰**：入口可见但不可点击，hover 提示"即将上线"
> - **空态**：页面/区块展示，但内容为空 + "暂无数据"
> - **仅预留数据结构**：数据库建字段但前端不展示，为后续迭代做准备
