## 3. UI 通用空态 / 错误态 / 降级态

> 页面级状态默认继承以下规则，有特殊需求在页面规格中覆盖。
> 后台和移动端的表现/文案不同，所以放端专属层。

| 场景 | 表现 | 默认文案 |
|------|------|----------|
| 列表无数据 | 居中插画 + 文案 + 操作引导 | {{文案}} |
| 搜索无结果 | 居中插画 + 文案 + 清除搜索建议 | {{文案}} |
| 请求失败（网络） | toast 提示 + 页面保留上次数据 | {{文案}} |
| 请求失败（服务端） | toast 提示 + 重试按钮 | {{文案}} |
| 无权限 | 页面不展示 / 操作按钮置灰 | {{文案}} |
| 第三方不可用（降级） | 功能入口置灰 + hover 说明 | {{文案}} |
| 字典加载失败 | 显示 code 原值 + toast 提示 | — |
| 数据为空（对象不存在） | 空态页面 + 返回入口 | {{文案}} |
