## 7. 状态与异常

> 穷举 6 类状态，不只 happy path。每类说明：触发 → 表现 → 用户可做什么。

| 状态类型 | 触发场景 | 页面表现 | 用户可做的操作 | 引用 |
|----------|----------|----------|---------------|------|
| 加载态 | 页面首次加载 / 切换 tab | 骨架屏 / spinner | 无 | — |
| 空态（无数据） | 列表返回 0 条 | 居中插画 + "暂无数据" | {{如：去创建}} | |
| 空态（搜索无结果） | 搜索/筛选后无匹配 | 居中插画 + "未找到相关内容" | 清除搜索条件 | |
| 错误态（网络） | 请求超时/断网 | toast + 保留上次数据 | 重试 | |
| 错误态（服务端） | 500/503 | toast + 错误信息 + 重试按钮 | 重试 | `GLB-ERR-xxx` |
| 无权限态 | 角色无权限 | 页面不展示 / 按钮置灰 | 无 | `GLB-ROLE-xxx` |
| 业务态-{{状态1}} | 对象状态 = {{xxx}} | {{该状态下的页面差异}} | {{可执行的操作}} | `GLB-SM-xxx` |

| 降级态 | 第三方不可用 | 功能入口置灰 + hover 说明 | 无 | `GLB-SRV-xxx` |
| 降级态 | 字典加载失败 | 显示 code 原值 + toast | 刷新页面 | |

## 8. 查询与列表（列表类页面填，详情/表单类可删除本节）

- **默认排序**：字段 + 方向
- **可选排序**：支持哪些字段排序
- **分页**：默认每页 N 条，支持切换 20/50/100
- **分页方式**：传统分页（页码） / 无限滚动 / 加载更多
- **列表轮询/实时刷新**：是否需要 / 间隔
- **批量选择**：是否支持勾选、全选、跨页选择
- **列表为空时的引导**：文案 + 操作按钮

## 9. 验收标准

> 每个核心功能至少一条 Given/When/Then。QA、AI、说明书共用。
> **不只写成功路径，异常路径也要有。**

```
AC-ID: {{端}}-{{模块号}}-AC-xxx
Given {{前置条件（数据 + 角色 + 页面状态），引用全局/模块/端专属 ID}}
When  {{操作步骤}}
Then  {{可观测结果（UI + 数据 + 副作用），引用状态机/错误码 ID}}
```
