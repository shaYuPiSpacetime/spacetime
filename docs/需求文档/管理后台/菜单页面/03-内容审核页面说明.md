# 内容审核页面说明

> 对应 PRD：`管理后台细化PRD-01_用户准入与资料认证初始化.md`
> 口径说明：本页面仅依据需求文档梳理，结合后台常规设计补充必要字段与交互，不受当前代码实现约束。

## 1. 页面范围

- 资料照片审核
- 文字内容审核

## 2. 列表字段

- 审核记录 ID
- 用户昵称
- 作者 UUID
- 内容类型
- 内容封面 / 缩略图
- 审核状态
- 提交时间
- 审核人
- 审核备注

## 3. 查询条件

- 用户 UUID
- 内容类型
- 审核状态
- 发布状态
- 关键字
- 时间范围

## 4. 操作按钮

- 通过
- 驳回
- 查看详情

## 5. 详情 / 弹窗

- 审核弹窗
- 内容详情弹窗

## 5.1 资料照片审核字段

- 审核记录 ID
- 用户 UUID
- 昵称
- 照片缩略图
- 照片类型
- 照片所属资料模块
- 提交时间
- 审核状态
- 驳回原因

## 5.2 文字内容审核字段

- 审核记录 ID
- 用户 UUID
- 昵称
- 文本字段类型
- 文本摘要
- 文本全文
- 提交时间
- 审核状态
- 驳回原因

## 6. 交互说明

- 审核通过或驳回后刷新列表。
- 资料照片和文字审核只影响对应内容，不直接改用户认证状态。
- 资料附加照片审核与头像认证审核必须分开存储状态、分开展示结果、分开记录驳回原因。
- 开放性文字审核与图片审核、认证审核状态不得混用。
- 详情弹窗用于查看原始内容、审核备注和驳回原因，不做批量操作。
