# 用户安全设置页面说明

> 对应 PRD：`管理后台细化PRD-06_认证与安全设置、我的页与搜索.md`
> 口径说明：本页面仅依据需求文档梳理，结合后台常规设计补充必要字段与交互，不受当前代码实现约束。

## 1. 页面范围

- 反馈箱
- 注销申请

## 2. 反馈箱字段

- 用户
- 反馈类型
- 内容
- 截图
- 联系方式
- 状态
- 处理备注
- 提交时间

### 2.1 反馈箱查询条件

- 用户 UUID
- 昵称
- 反馈类型
- 处理状态
- 提交时间范围

### 2.2 反馈详情字段

- 用户 UUID
- 昵称
- 反馈类型
- 内容
- 截图
- 联系方式
- 状态
- 处理备注
- 提交时间
- 处理人
- 处理时间

## 3. 注销申请字段

- 用户
- 状态
- 后悔期结束时间
- 阻断原因
- 注销原因
- 后台备注
- 申请时间
- 撤销时间

### 3.1 注销申请查询条件

- 用户 UUID
- 昵称
- 注销状态
- 是否存在阻断原因
- 申请时间范围

### 3.2 注销申请详情字段

- 用户 UUID
- 昵称
- 状态
- 后悔期结束时间
- 阻断原因
- 注销原因
- 后台备注
- 申请时间
- 撤销时间
- 最终注销时间

## 4. 操作按钮

- 查询
- 重置
- 查看详情
- 保存
- 处理状态切换
- 补充备注

## 5. 详情 / 弹窗

- 反馈详情弹窗
- 注销申请详情弹窗
- 处理状态选择
- 备注输入框
- 阻断原因展示区

## 6. 交互说明

- 反馈箱仅支持查询、查看与处理状态更新。
- 注销申请支持查看详情并补充后台备注和阻断原因。
- 注销申请详情需展示后悔期、阻断校验项和最终清除时间。
- 详情弹窗保存后刷新列表。
- 反馈状态建议按 `待处理 → 处理中 → 已解决 / 已关闭` 流转。
- 注销申请状态建议保留 `后悔期 / 已撤销 / 已注销 / 已阻断` 的完整历史。
