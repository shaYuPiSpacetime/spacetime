## 9. 交互说明

- 订单按 `VIP` / `成家币` 分类。
- 支付取消后的订单应进入关闭态，不允许手工改为支付成功。
- 退款管理支持查询退款中和已退款订单。
- 处理退款前需二次确认。
- 详情页需展示完整支付与回调信息。
- 流水需区分 `充值所得 / 赠送所得 / 退款退回 / 消费扣减` 等来源。
- 轻量对账以按日汇总核对为主，首版不展开完整渠道账单上传与差错追账系统。
