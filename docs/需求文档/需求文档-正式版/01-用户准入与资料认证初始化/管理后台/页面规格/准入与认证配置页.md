## 2. 配置项

| 配置 ID | 配置项 | 默认值 | 高风险 | 生效范围 |
|---------|--------|--------|--------|----------|
| `M01-CFG-age-range` | 年龄允许范围 | 18-60 | 是 | 出生日期页、准入拦截 |
| `M01-CFG-required-fields` | 用户字段展示/必填配置 | 以 `M01-DATA-user-input-fields` 为字段池 | 是 | 轻量引导、基本资料、编辑资料页面组 |
| `M01-CFG-profile-score-weight` | 资料完整度权重 | 参与字段、在校生/职场人分值均由后台配置，总分 100 | 是 | 资料页、推荐加权 |
| `M01-CFG-core-access-threshold` | 核心准入门槛 | 三重认证通过 | 是 | 曝光、喜欢、私信、社区 |
| `M01-CFG-photo-upload-limit` | 图片上传限制 | 学历最多 4 张，相册最多 9 张，资料背景图最多 1 张 | 否 | 头像/学历/相册/资料背景图 |
| `M01-CFG-field-validation` | 字段校验规则 | 见公共定义 7.1 | 否 | 资料、认证、验证码 |
| `M01-CFG-text-length` | 开放文本长度 | 关于我 20-300 字 | 否 | 自我介绍、开放文本 |
| `M01-CFG-sms-login-limit` | 短信验证码规则 | 60 秒倒计时、5 分钟有效、每日 10 次 | 是 | 登录授权页 |
| `M01-CFG-voice-safety` | 语音介绍内容安全 | 微信音频机审通过后展示 | 是 | 语音介绍页 |
| `M01-CFG-education-sla` | 学历审核 SLA | 24 小时 | 否 | 学历审核页 |
| `M01-CFG-copy` | 准入/认证文案 | 准入拦截、认证提示、协议、驳回模板、异常文案、内容安全文案 | 否 | APP 文案展示 |

## 4. 规则

1. 年龄范围、用户字段展示/必填配置、核心准入门槛、资料完整度权重属于高风险配置，保存需二次确认。
2. 核心准入默认必须实名 + 头像 + 学历三项通过，不允许默认降级为单实名。
3. 用户字段配置只能从 `M01-DATA-user-input-fields` 字段池中选择；配置变更只影响变更后的判定，历史资料按原始冗余值展示；准入状态可按最新配置重算。
4. 修改文案不得改变产品已确认口径，例如“身份证号”“千寻币”“三重认证”。
5. 语音介绍只配置微信音频机审策略和提示文案，首版不提供单独人工语音审核开关。
