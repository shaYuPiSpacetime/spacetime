## 1. 页面定位

- **目标用户**：运营、超级管理员
- **核心任务**：维护用户准入和资料认证所需字典
- **页面类型**：字典管理页

---

## 2. 字典范围

| 字典类型 | 用途 | 规则 |
|----------|------|------|
| 学历 | 轻量引导、基本资料、学历认证 | 系统字典启用项展示 |
| 职业 | 基本资料职场人字段 | 支持手动填写兜底 |
| 年收入 | 基本资料职场人字段 | 系统字典启用项展示 |
| 婚姻状态 | 基本资料 | 不等于婚况认证 |
| 地区 | 现居地、家乡 | 不支持海外国家/地区 |
| 学校 | 学历认证、在校生资料 | 先查本地冗余，未命中调用全国大学院校查询 API 后入库 |
| 标签 | 扩展资料、资料完整度 | 支持启停和排序 |

## 4. 操作表

| 操作 ID | 操作名 | 前置权限 | 二次确认 | 成功态 | 失败态 |
|---------|--------|----------|----------|--------|--------|
| `ADM-01-PAGE-access-dict-ACT-create` | 新增字典项 | 字典编辑权限 | 否 | 新增启用/停用项 | 编码重复提示 |
| `ADM-01-PAGE-access-dict-ACT-edit` | 编辑字典项 | 字典编辑权限 | 已被使用项是 | 保存并记录日志 | 校验失败提示 |
| `ADM-01-PAGE-access-dict-ACT-disable` | 停用 | 字典编辑权限 | 是 | 新增选择不再展示，历史不变 | 已被系统锁定项不可停用 |
| `ADM-01-PAGE-access-dict-ACT-sort` | 排序 | 字典编辑权限 | 否 | 移动端按新顺序展示 | 保存失败回滚 |
| `ADM-01-PAGE-access-dict-ACT-sync-school` | 同步学校 | 字典编辑权限 | 是 | 调用全国大学院校查询 API 并入库 | 服务不可用提示稍后重试 |

---

## 5. 规则

1. 字典停用/改名只影响新增数据。
2. 历史用户资料按提交时冗余值展示，不跟随字典名称变化。
3. 学校字典优先查本地冗余；未命中时调用全国大学院校查询 API，再写入本地。
4. 地区字典首版仅支持中国大陆。
5. 婚姻状态只是资料字段，不产生婚况认证。
