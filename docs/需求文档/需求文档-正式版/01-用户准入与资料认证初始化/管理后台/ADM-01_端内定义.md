## 2. 权限矩阵

角色列与管理后台全局角色基线对应：运营=`GLB-ROLE-operation`，客服=`GLB-ROLE-customer-service`，审核员=`GLB-ROLE-auditor`，风控=`GLB-ROLE-risk`，超级管理员=`GLB-ROLE-super-admin`。

| 权限 ID | 权限名称 | 运营 | 客服 | 审核员 | 风控 | 超级管理员 |
|---------|----------|------|------|--------|------|------------|
| `ADM-01-PERM-user-view` | 查看 App 用户列表/详情 | 是 | 是 | 是 | 是 | 是 |
| `ADM-01-PERM-sensitive-view` | 查看完整手机号/身份证/姓名 | 否 | 按授权 | 否 | 按授权 | 是 |
| `ADM-01-PERM-gender-edit` | 客服修改性别 | 否 | 是 | 否 | 否 | 是 |
| `ADM-01-PERM-account-status-edit` | 冻结/解冻、启用/停用 | 否 | 是 | 否 | 是 | 是 |
| `ADM-01-PERM-cert-audit` | 实名/头像/学历审核 | 否 | 否 | 是 | 是 | 是 |
| `ADM-01-PERM-content-audit` | 资料图片/文字审核 | 是 | 否 | 是 | 是 | 是 |
| `ADM-01-PERM-config-edit` | 修改准入配置 | 否 | 否 | 否 | 否 | 是 |
| `ADM-01-PERM-export` | 导出用户/审核数据 | 否 | 否 | 否 | 按授权 | 是 |

## 3. 列表与搜索通用规则

| 对象 | 默认排序 | 默认分页 | 搜索范围 | 审核后刷新 |
|------|----------|----------|----------|------------|
| App 用户卡片 | 创建时间倒序 | 每页 10 条 | 用户姓名、用户昵称、手机号、身份证号、标签 | 不适用 |
| 实名/头像/学历审核列表 | 提交时间倒序；可叠加待审核优先筛选 | 每页 10 条 | 用户姓名、用户昵称、手机号、身份证号、标签 | 通过/驳回后实时刷新当前页和统计 |
| 资料图片/开放文字审核列表 | 提交时间倒序；可叠加待审核优先筛选 | 每页 10 条 | 用户姓名、用户昵称、手机号、身份证号、标签 | 通过/驳回后实时刷新当前页和统计 |

高敏字段搜索只用于匹配查询，列表默认仍按权限展示脱敏值；无查看高敏权限时不得在前端明文回显手机号、身份证号、姓名。

## 5. 审计清单

| 操作 | 是否二次确认 | 是否记录审计 | 审计字段 |
|------|--------------|--------------|----------|
| 查看完整姓名/身份证号/手机号 | 是 | 是 | 操作人、用户、字段、时间、理由 |
| 下载实名/学历/头像原图 | 是 | 是 | 操作人、材料类型、时间、理由 |
| 修改性别 | 是 | 是 | 变更前后值、操作人、原因、时间 |
| 冻结/解冻、启用/停用 | 是 | 是 | 目标用户、动作、原因、时间 |
| 实名/头像/学历审核通过或驳回 | 是 | 是 | 审核对象、结果、驳回原因、时间 |
| 资料图片/文字审核通过或驳回 | 是 | 是 | 审核对象、结果、驳回原因、时间 |
| 修改准入配置/字典 | 是 | 是 | 配置项、变更前后值、时间 |
| 导出数据 | 是 | 是 | 导出范围、字段、筛选条件、下载时间、下载人 |

## 6. 管理后台 UI 状态

| 场景 | 表现 | 默认文案 |
|------|------|----------|
| App 用户列表无数据 | 卡片宫格空态 | 暂无 App 用户 |
| 搜索无结果 | 保留筛选栏 + 空态 | 未找到相关用户 |
| 审核列表无数据 | 表格空态 | 暂无待审核记录 |
| 无权限 | 菜单隐藏或按钮置灰 | 无操作权限 |
| 敏感字段无权限 | 展示脱敏值 | 无完整查看权限 |
| 第三方学历服务不可用 | 审核记录保留审核中/待人工 | 第三方认证服务暂不可用 |
| 用户材料缺失 | 详情区展示缺失态 | 材料缺失，请联系用户重新提交 |
