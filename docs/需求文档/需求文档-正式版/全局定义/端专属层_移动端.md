# 全局定义（端专属层）- 移动端

> 本文件是移动端正式端专属层母文件，统一登记移动端全局通用的信息架构原则、登录态基线、UI 通用态与端级文案规范。
> 单模块页面树、模块内登录态差异、模块端内 UI 状态放在对应模块的移动端端内定义中，不放本文；跨端或多页面复用的模块文案 key 放模块公共定义。
> 跨模块项目级业务定义在 `共享层_项目级.md`；单模块跨端业务定义在对应模块公共定义中。

## 1. 适用范围与归属边界

| 内容类型 | 放置位置 | 说明 |
|----------|----------|------|
| 移动端全局导航原则、登录态基线、通用空态/错误态 | 本文件 | 对全部移动端模块默认生效 |
| 单模块页面树、入口位置、端内 UI 状态 | 各模块移动端端内定义 | 例如 `APP-07_端内定义.md` |

| 跨模块业务术语/规则/错误码 | `共享层_项目级.md` | 使用 `GLB-*` |
| 单模块跨端业务定义 | 模块公共定义 | 使用 `Mxx-*` |

## 2. 移动端登录态基线

| 用户状态 | 默认能力 | 说明 |
|----------|----------|------|
| 未登录 | 可浏览公开内容；涉及个人数据/资产/互动的操作需登录 | 具体引导方式由页面规格定义 |
| 已登录普通用户 | 可使用当前账号授权范围内能力 | 模块如有额外限制，在模块端内定义或页面规格补充 |

---

## 3. UI 通用空态 / 错误态 / 降级态

| 场景 | 表现 | 默认文案 |
|------|------|----------|
| 请求失败（网络） | toast + 保留上次数据 | 网络开小差了，请稍后重试 |
| 请求失败（服务端） | toast + 重试按钮 | 加载失败，点击重试 |
| 无权限 / 未登录 | 引导登录或展示无权限提示 | 请先登录 |
| 列表无数据 | 居中插画 + 文案 + 可选操作引导 | 暂无数据 |
| 第三方能力不可用 | 降级入口 + 可重试提示 | 暂不可用，请稍后重试 |
