## 1. 页面定位

- **目标用户**：运营、超级管理员
- **核心任务**：配置会员权益/套餐、千寻币套餐、千寻币消费场景、保留期、社交与订单参数和曝光包预留；会员权益按 UI 固定为心动名单、访客、免费悄悄话、额外浏览、高级筛选、曝光、隐私权益、三天回放、每日心动机会；会员权益和千寻币消费场景均维护移动端图标；不配置 SVIP/高端服务、积分公式/规则/记录，也不提供连续订阅状态同步
- **页面类型**：多 Tab 配置页

## 2. 布局

```text
Tab: 会员权益 | 会员套餐 | 千寻币套餐 | 千寻币消费场景 | 解锁保留期 | 社交与订单参数 | 曝光包预留
Tab 内容配置表/表单 -> 底部固定保存按钮（二次确认） -> 右侧变更日志抽屉
```

| 区块 | 位置 | 内容 | 是否可折叠 | 是否记住展开状态 |
|------|------|------|-----------|----------------|
| 会员权益 Tab | Tab1 | 系统内置 9 项权益启停和移动端图标配置；免费悄悄话/额外浏览/每日心动机会配置次数，曝光配置分数；名称、类型、说明只读；不提供排序 | 否 | 是 |
| 会员套餐 Tab | Tab2 | 普通套餐/连续订阅套餐、原价、优惠价、时长/周期、标签、上架状态、协议状态；不含 SVIP/高端服务 | 否 | 是 |
| 千寻币套餐 Tab | Tab3 | 千寻币套餐、原价、优惠价、到账币数、赠送币数、标签、推荐档、上架状态 | 否 | 是 |
| 千寻币消费场景 Tab | Tab4 | 6 个千寻币消费场景的单价、启停和移动端图标配置，不包含 `invite_*` 奖励场景 | 否 | 是 |
| 解锁保留期 Tab | Tab5 | 理想型批量上限、理想型保留天数、觅缘回看解锁保留天数 | 否 | 是 |
| 社交与订单参数 Tab | Tab6 | 普通/会员每日查看配额、会员到期提醒提前天数、退款展示开关；未支付订单固定 30 分钟仅做说明 | 否 | 是 |
| 曝光包预留 Tab | Tab7 | 曝光包场景占位开关与说明 | 否 | 是 |

| `ADM-04-CFG-06` | 商业化配置-社交与订单参数 Tab | 配额、到期提醒、退款展示开关、未支付订单 30 分钟固定说明 | 承接原孤儿配置项；订单关闭时长不提供编辑 |
| `ADM-04-CFG-07` | 商业化配置-曝光包预留 Tab | 预留开关、说明文案、不可售卖提示 | 首版默认关闭 |
| `ADM-04-CFG-08` | 会员套餐新增/编辑弹窗 | 套餐名称、套餐类型、原价、优惠价、时长/周期、标签、状态、协议/微信商品校验 | 套餐类型仅允许普通套餐、连续订阅套餐；不允许 SVIP/高端服务 |
| `ADM-04-CFG-08B` | 千寻币套餐新增/编辑弹窗 | 套餐类型固定千寻币套餐、原价、优惠价、到账币数、赠送币数、标签、是否推荐、状态 | 推荐档最多 1 个 |

| `ADM-04-CFG-10` | 配置变更日志抽屉 | 操作人、变更字段、变更前后值、时间、备注 | 从页面右上角进入 |

### 2.2 编辑控件口径

| 配置区域 | 展现形式 | 编辑控件 | 新增/删除方式 | 保存方式 |
|----------|----------|----------|----------------|----------|
| 会员权益 | 固定表格 | 移动端图标选择/输入；启停开关；免费悄悄话/额外浏览/每日心动机会次数输入；曝光分数输入 | 不允许新增、删除、排序；名称、类型、说明均只读 | 底部统一保存 |
| 会员套餐 | 列表表格 | 编辑弹窗、状态开关、标签多选 | 新增/编辑均打开弹窗；下架需二次确认；不允许新增 SVIP/高端服务类型 | 底部统一保存 |
| 千寻币套餐 | 列表表格 | 编辑弹窗、数字输入、推荐单选 | 新增/编辑均打开弹窗；推荐档互斥 | 底部统一保存 |
| 积分管理 | 不展示 | 不提供 | demo 历史入口不属于 PRD-04 首版 | 不提供 |
| 千寻币消费场景 | 固定场景表格 | 移动端图标选择/输入、数字输入、启停开关 | 不允许新增场景；场景由 `M04-ENUM-coin-biz-scene` 消费场景限定 | 底部统一保存 |
| 解锁保留期 | 表单 | 数字输入、步进器 | 不涉及新增 | 底部统一保存 |
| 社交与订单参数 | 表单 | 数字输入、开关、单位选择 | 不涉及新增 | 底部统一保存 |
| 曝光包预留 | 表单 | 开关、说明文本域 | 不涉及新增 | 底部统一保存 |

| `ADM-04-PAGE-commerce-config-ACT-save` | 保存配置 | 底部固定 | 表单校验通过 | `ADM-04` 配置权限 | 是，提示将立即影响移动端 | 保存成功，配置缓存失效，写审计日志 | `M04-ERR-config-missing` 或字段校验错误 |
| `ADM-04-PAGE-commerce-config-ACT-view-log` | 查看变更日志 | 右上角 | 有页面查看权限 | `ADM-04` 查看权限 | 否 | 打开变更日志抽屉 | 审计服务异常时提示稍后重试 |

## 9. 验收标准

```text
AC-ID: ADM-04-AC-config-save
Given 运营具备配置权限，修改一个千寻币套餐的到账币数
When  点击保存并确认二次弹窗
Then  M04-CFG-coin-package-list 更新，移动端配置缓存失效，审计日志记录操作人、变更前后值和时间

AC-ID: ADM-04-AC-config-subscription-block
Given 连续订阅套餐未绑定微信连续订阅商品或协议
When  运营尝试上架连续订阅套餐
Then  保存被阻断，提示补齐微信连续订阅商品、周期和协议配置

AC-ID: ADM-04-AC-config-benefit-readonly
Given 运营进入会员权益 Tab
When  查看心动名单、访客、免费悄悄话、额外浏览、高级筛选、曝光、隐私权益、三天回放、每日心动机会
Then  名称、类型、说明均不可编辑，不展示新增、删除、排序控件；每项展示移动端图标配置；心动名单、访客、高级筛选、隐私权益、三天回放只有开关，免费悄悄话、额外浏览、每日心动机会额外展示每日次数输入，曝光额外展示分数输入

AC-ID: ADM-04-AC-config-scene-icon
Given 运营进入千寻币消费场景 Tab
When  查看 6 个消费场景
Then  每个场景展示场景 code、移动端图标配置、单价、启停和影响页面；不展示 `invite_*` 奖励场景；保存后千寻币充值页消费场景列表按配置图标展示

AC-ID: ADM-04-AC-config-scope-boundary
Given 历史 demo 存在积分管理、SVIP 或高端服务配置入口
When  运营进入商业化配置页
Then  页面不展示这些入口；会员套餐只允许普通套餐/连续订阅套餐，千寻币为唯一付费虚拟币口径
```
