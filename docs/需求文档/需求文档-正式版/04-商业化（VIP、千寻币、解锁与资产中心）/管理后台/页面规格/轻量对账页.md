## 1. 页面定位

- **目标用户**：财务、超级管理员
- **核心任务**：按日查看会员订单、千寻币充值订单、退款订单数量与金额；首版不统计 SVIP、高端服务、积分或其他付费虚拟币收入
- **页面类型**：轻量统计页

---

```text

```

| 日期筛选 | 页面顶部 | 日期范围、订单类型、查询、导出 | 否 | 是 |

| 日维度表格 | 页面中部 | 按日聚合的订单数、金额、退款、净收入 | 否 | 是 |

## 6. 数据联动规则

| 触发字段 | 触发事件 | 影响字段 | 联动行为 | 备注 |
|----------|----------|----------|----------|------|
| 日期范围 | 查询 | 汇总卡片/日维度表格 | 按所选日期重新聚合，并刷新数据更新时间 | |
| 订单类型 | 切换 | 汇总卡片/日维度表格 | 全部时展示会员+千寻币；单类型时对应列保留、无关列显示 0 或 `-` | |
| 订单类型 | 返回 SVIP/高端服务/积分订单 | 汇总卡片/日维度表格 | 不纳入 PRD-04 首版统计口径，不展示独立列 | `M04-RULE-member-tier-scope` / `M04-RULE-paid-coin-scope` |
| 退款金额 | 统计 | 净收入 | 净收入=支付金额-退款金额，可为负 | |
| 导出 | 确认 | 导出任务 | 导出当前筛选口径与列字段，写审计 | |

## 7. 状态与异常

| 状态类型 | 触发场景 | 页面表现 | 用户可做的操作 | 引用 |
|----------|----------|----------|---------------|------|
| 加载态 | 查询统计 | 卡片/表格 loading | 无 | — |
| 空态 | 日期内无数据 | 表格空态，汇总为 0 | 调整日期 | — |
| 错误态 | 查询失败 | toast + 重试 | 重试 | — |
| 无权限态 | 非财务/超管访问 | 菜单隐藏或无权限态 | 无 | ADM-04 端内定义 |
| 业务态-正常 | 有支付/退款数据 | 汇总卡和表格展示 | 导出 | — |
| 降级态 | 统计任务延迟 | 展示数据更新时间 | 稍后刷新 | — |
| 边界态 | 出现 SVIP/高端服务/积分收入 | 不纳入汇总，不展示独立统计列 | 调整筛选 | `M04-RULE-member-tier-scope` / `M04-RULE-paid-coin-scope` |

---

## 8. 查询与列表

- **默认排序**：日期倒序。
- **分页**：首版按日期聚合，默认不分页；当日期范围超过 31 天时可分页或限制查询范围，由技术方案确认。
- **导出**：异步导出，导出口径与当前筛选条件一致。

## 9. 验收标准

```text
AC-ID: ADM-04-AC-reconcile-daily
Given 财务选择最近 7 天
When  点击查询
Then  页面按日展示会员订单数、千寻币订单数、支付金额、退款金额和净收入，并展示数据更新时间

AC-ID: ADM-04-AC-reconcile-scope
Given 历史 demo 存在积分收入或高端服务收入口径
When  财务进入轻量对账页查询商业化收入
Then  页面只统计时空邂逅会员与千寻币订单，不展示 SVIP、高端服务、积分或其他付费虚拟币统计列
```
