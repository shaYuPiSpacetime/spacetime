# ADM-04 端内定义 - 商业化管理

> 本文件登记 PRD-04 在管理后台内专属的信息架构、权限矩阵、敏感操作审计与 UI 状态。
> 管理后台全局通用规则见 `../../全局定义/端专属层_管理后台.md`；PRD-04 跨端业务定义见 `../PRD-04_模块公共定义.md`。
> 对齐旧版后台 PRD-04：不新增独立一级“商业化运营工作台”，能力分布在用户管理中心、财务中心、运营中心、系统管理中心和移动端配置管理。

| 版本 | 日期 | 修改人 | 变更摘要（改动须列出受影响的页面 ID） |
|------|------|--------|----------|
| 版本04 | 2026-06-30 | Codex | 明确 `ADM-04-PAGE-user-asset-detail` 由用户管理详情承载不作为独立菜单页；同步订单详情发起退款和退款记录默认已退款，影响 `ADM-04-PAGE-user-asset-detail`、`ADM-04-PAGE-commerce-order-list`、`ADM-04-PAGE-refund-list` |
| 版本03 | 2026-06-30 | Codex | 同步未支付订单 30 分钟固定关闭和退款页仅查询展示，影响 `ADM-04-PAGE-commerce-config`、`ADM-04-PAGE-refund-list` |
| 版本02 | 2026-06-26 | Codex | 同步商业化配置页社交与订单参数能力，影响 `ADM-04-PAGE-commerce-config` |
| 版本01 | 2026-06-25 | Codex | 版本 01：由旧版管理后台 PRD-04 转写正式版端内定义 |

## 1. 信息架构（菜单 / 页面树）

| 一级菜单 | 二级页面 / 区块 | 页面 ID | 一句话职责 | 优先级 |
|----------|----------------|---------|------------|--------|
| 用户管理中心 | 用户详情弹窗/详情页-商业化内容区 | `ADM-04-PAGE-user-asset-detail` | 在用户管理详情内查看资产摘要、会员订单、千寻币流水、解锁记录、退款记录；不新增 ADM-04 独立菜单页 | P0 |
| 财务中心 | 商业化订单管理 | `ADM-04-PAGE-commerce-order-list` | 查询会员订单和千寻币充值订单，支持筛选、详情、导出和订单内发起退款 | P0 |
| 财务中心 | 资产流水管理 | `ADM-04-PAGE-asset-flow-list` | 查询充值、消费、赠送、奖励、退款等资产流水 | P0 |
| 财务中心 | 退款记录管理 | `ADM-04-PAGE-refund-list` | 查询订单详情发起后的退款记录、发起人、原因、资产回退与前台展示状态 | P1 |
| 财务中心 | 轻量对账 | `ADM-04-PAGE-commerce-reconcile` | 按日查看订单、金额、退款的轻量统计 | P1 |
| 移动端配置管理 | 商业化配置 | `ADM-04-PAGE-commerce-config` | 配置会员权益/套餐、千寻币套餐、单价、保留期、社交与订单参数、曝光包预留 | P0 |
| 运营中心 | 文案与消息中心 | `ADM-GLB-PAGE-copy-message-center` | 配置会员页、充值页、支付结果、退款状态等文案与通知模板 | P0 |

## 2. 角色与权限矩阵

### 2.1 角色引用

| 角色 code | 显示名 | 本模块中承担的职责 |
|-----------|--------|-------------------|
| `GLB-ROLE-operation` | 运营 | 配置会员/千寻币套餐、权益、文案，查看商业化数据 |
| `GLB-ROLE-customer-service` | 客服 | 查看用户资产、订单、流水与解锁记录，协助排查客诉 |
| `GLB-ROLE-finance` | 财务/结算 | 查看订单、资产流水、退款与轻量对账，执行导出 |
| `GLB-ROLE-risk` | 风控 | 查看异常资产、退款、人工调整风险记录 |
| `GLB-ROLE-super-admin` | 超级管理员 | 修改全部配置、查看高敏信息、执行受控人工修正 |

### 2.2 权限矩阵（ADM-04 涉及操作）

| 操作 / 能力 | 运营 | 客服 | 财务 | 风控 | 超管 |
|-------------|------|------|------|------|------|
| 查看用户资产摘要 | 允许 | 允许 | 允许 | 允许 | 允许 |
| 查看完整手机号/支付定位信息 | 禁止 | 按授权 | 按授权 | 按授权 | 允许 |
| 查看商业化订单 | 允许 | 允许 | 允许 | 允许 | 允许 |
| 导出订单/流水/退款 | 禁止 | 禁止 | 允许 | 按授权 | 允许 |
| 配置会员权益/套餐 | 允许 | 禁止 | 禁止 | 禁止 | 允许 |
| 配置千寻币套餐/单价 | 允许 | 禁止 | 禁止 | 禁止 | 允许 |
| 配置退款前台展示 | 禁止 | 禁止 | 允许 | 禁止 | 允许 |
| 查看特批退款记录 | 禁止 | 只读 | 允许 | 按授权 | 允许 |
| 手工补偿/人工调整资产 | 禁止 | 禁止 | 禁止 | 禁止 | 允许 |
| 配置商业化文案/通知 | 允许 | 禁止 | 禁止 | 禁止 | 允许 |

### 2.3 敏感操作审计要求

| 操作 | 是否二次确认 | 是否记录审计日志 | 审计日志保留 |
|------|-------------|-----------------|-------------|
| 修改会员权益/套餐/价格 | 是 | 是 | >= 1 年 |
| 修改千寻币套餐/单价 | 是 | 是 | >= 1 年 |
| 修改退款展示等高风险参数 | 是 | 是 | >= 1 年 |
| 查看完整手机号或支付相关敏感信息 | 是 | 是 | >= 1 年 |
| 导出订单/流水/退款 | 是 | 是 | >= 1 年 |
| 订单详情内发起退款 | 是 | 是 | >= 1 年 |
| 手工补偿/人工调整资产 | 是 | 是 | >= 1 年 |

### 2.4 后台展示标识规则

| 对象 | 后台展示标识 | 说明 |
|------|--------------|------|
| 用户 | 用户 UUID + 昵称/手机号（按权限脱敏） | 不直接展示数据库内部主键 |
| 订单 | 商业化订单号 `ORD-yyyymmdd-xxxx` | 路由可使用内部 id |
| 千寻币流水 | 流水号 `AFL-yyyymmdd-xxxx` | 用户侧可不展示完整流水号 |
| 退款记录 | 退款单号 `RFD-yyyymmdd-xxxx` | 与订单号关联展示 |
| 解锁记录 | 解锁记录号 `ULK-yyyymmdd-xxxx` | 用于客服排查 |

## 3. UI 空态 / 错误态 / 降级态

| 场景 | 表现 | 默认文案 |
|------|------|----------|
| 订单列表无数据 | 表格空态 | 暂无订单记录 |
| 资产流水无数据 | 表格空态 | 暂无资产流水 |
| 配置项未初始化 | 配置页提示需先填写 | 商业化配置未初始化 |
| 无权限 | 菜单隐藏或按钮置灰 | 无操作权限 |
| 支付渠道数据不可用 | 订单详情保留本地订单状态并提示渠道数据延迟 | 支付渠道数据暂不可用 |
| 导出生成中 | loading + 异步下载提示 | 导出生成中，请稍后在下载中心查看 |
