## 1. 页面定位

- **目标用户**：已开通或曾开通过连续订阅套餐的已登录用户
- **核心任务**：查看当前连续订阅状态、下次续费信息、协议说明，并找到取消续费入口或取消指引
- **页面类型**：订阅管理页

---

## 2. 布局

```text
订阅状态卡 -> 套餐与续费信息 -> 协议与扣费说明 -> 取消续费入口/指引 -> 订单记录入口
```

| 区块 | 位置 | 内容 | 是否可折叠 | 是否记住展开状态 |
|------|------|------|-----------|----------------|
| 订阅状态卡 | 顶部 | 连续订阅状态、会员状态、到期时间 | 否 | 否 |
| 套餐与续费信息 | 中部 | 套餐名称、价格、周期、下次续费时间 | 否 | 否 |
| 协议与扣费说明 | 中部 | 会员协议、连续订阅协议、扣费说明 | 否 | 否 |
| 取消续费入口/指引 | 底部 | 打开微信订阅管理入口或展示取消步骤 | 否 | 否 |
| 订单记录入口 | 底部 | 查看会员订单记录 | 否 | 否 |

---

## 4. 字段表

| 字段 ID | 显示名 | 类型 | 必填 | 取值范围 | 校验规则 | 默认值 | 可编辑 | 敏感级别 | 数据来源 |
|---------|--------|------|------|----------|----------|--------|--------|----------|----------|
| `APP-04-PAGE-subscription-manage-FIELD-subscription-status` | 连续订阅状态 | enum/string | 是 | 生效中/已取消/未开通/暂不可用 | 按服务返回展示中文 | 未开通 | 否 | 普通 | 用户会员订阅状态 |
| `...-FIELD-vip-status` | 会员状态 | enum | 是 | `M04-ENUM-vip-status` | 中文展示 | `inactive` | 否 | 普通 | 用户资产摘要 |
| `...-FIELD-package-name` | 套餐名称 | string | 否 | 套餐快照 | 无订阅展示 `-` | 无 | 否 | 普通 | 订阅订单快照 |
| `...-FIELD-renew-price` | 续费金额 | decimal | 否 | ≥0 | 展示人民币金额 | 无 | 否 | 普通 | 订阅订单快照 |
| `...-FIELD-renew-cycle` | 续费周期 | string | 否 | 月/季/年等 | 取套餐配置 | 无 | 否 | 普通 | `M04-CFG-subscription-package-list` |
| `...-FIELD-next-renew-time` | 下次续费时间 | datetime | 否 | `yyyy-MM-dd HH:mm:ss` | 已取消或服务暂不可用展示 `-` | 无 | 否 | 普通 | 微信连续订阅/本地订阅记录 |
| `...-FIELD-cancel-guide` | 取消续费指引 | string/json | 是 | 文案/步骤 | 微信取消入口不可用时必须展示平台内指引 | 默认指引 | 否 | 普通 | `M04-SRV-wechat-subscription` |

## 6. 数据联动规则

| 触发字段 | 触发事件 | 影响字段 | 联动行为 | 备注 |
|----------|----------|----------|----------|------|
| 连续订阅状态 | 生效中 | 取消续费入口 | 展示取消续费按钮和下次续费时间 | |
| 连续订阅状态 | 已取消 | 取消续费入口 | 隐藏取消按钮，展示“已取消，会员有效期内仍可使用” | |
| 连续订阅服务 | 不可用 | 页面内容 | 展示暂不可用说明和普通会员续费入口 | `M04-SRV-wechat-subscription` |
| 取消续费 | 成功返回或用户手动完成 | 订阅状态卡 | 刷新订阅状态，不影响已支付会员有效期 | |

## 7. 状态与异常

| 状态类型 | 触发场景 | 页面表现 | 用户可做的操作 | 引用 |
|----------|----------|----------|---------------|------|
| 加载态 | 首次加载订阅状态 | 骨架屏 | 无 | — |
| 空态 | 无连续订阅记录 | 展示未开通说明和去开通会员入口 | 去会员中心 | `M04-TERM-subscription` |
| 错误态 | 状态查询失败 | toast + 重试 | 重试 | — |
| 无权限态 | 未登录进入 | 引导登录 | 登录 | APP-04 端内定义 |
| 业务态-订阅生效 | 状态为生效中 | 展示下次续费信息和取消续费入口 | 取消续费/查看协议 | `M04-SRV-wechat-subscription` |
| 业务态-已取消 | 状态为已取消 | 展示已取消说明和会员有效期 | 查看订单 | |
| 降级态 | 微信连续订阅服务不可用 | 展示暂不可用说明，不展示真实取消按钮 | 返回/选择普通套餐 | `M04-ERR-subscription-unsupported` |

## 9. 验收标准

```text
AC-ID: APP-04-AC-subscription-manage-active
Given 用户存在生效中的连续订阅关系
When  从会员中心进入连续订阅管理页
Then  页面展示订阅状态、套餐名称、下次续费时间、协议入口和取消续费入口

AC-ID: APP-04-AC-subscription-cancel-guide
Given 微信连续订阅服务异常或取消入口不可用
When  用户进入连续订阅管理页
Then  页面展示暂不可用说明或取消续费指引，不展示不可执行的真实取消按钮
```

### 验收标准清单

| AC ID | 场景 | 类型 | 优先级 |
|-------|------|------|--------|
| `APP-04-AC-subscription-manage-active` | 连续订阅状态可查 | 正常 | P0 |
| `APP-04-AC-subscription-cancel-guide` | 取消续费降级指引 | 异常 | P0 |
