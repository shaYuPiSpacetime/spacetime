## 3. 筛选与搜索

### 3.1 搜索

| 项 | 说明 |
|----|------|
| 搜索字段 | 奖励流水号、邀请人 UUID、被邀请人 UUID、邀请人手机号、被邀请人手机号 |
| 搜索方式 | 精确匹配 |
| 搜索触发 | 点击查询 |
| 清空搜索 | 有清空按钮 |

### 3.2 筛选条件

| 筛选 ID | 筛选名 | 类型 | 选项来源 | 是否多选 | 默认值 | 是否可清除 |
|---------|--------|------|----------|----------|--------|-----------|
| `...-FILTER-event` | 奖励事件类型 | 下拉 | 注册成功登录奖励/资料完善奖励/认证完成奖励/首次会员奖励/首次充值奖励（来源：`M07-ENUM-invite-reward-event`） | 是 | 全部 | 是 |
| `...-FILTER-status` | 奖励状态 | 下拉 | 待发放/已到账/冻结中/无效（来源：`M07-ENUM-invite-reward-status`） | 是 | 全部 | 是 |
| `...-FILTER-time` | 时间范围 | 日期范围 | — | — | 无 | 是 |

### 3.3 筛选交互

- 筛选项变化后：点击「查询」触发
- 显示当前筛选标签：是
- URL 持久化：是

- **默认排序**：创建时间倒序

## 9. 验收标准

```
AC-ID: ADM-07-AC-reward-list-filter
Given 存在多种奖励事件与状态的流水
When  筛选奖励事件=认证完成奖励 且 奖励状态=已到账
Then  仅展示匹配记录，每条含奖励流水号、双方用户 UUID/昵称/手机号、币数、到账时间

AC-ID: ADM-07-AC-reward-list-export
Given 具备导出权限的后台用户设置时间范围
When  点击导出
Then  按筛选异步生成流水明细，下载中心可下载；导出字段包含手机号且不脱敏，操作记审计
```

### 验收标准清单

| AC ID | 场景 | 类型 | 优先级 |
|-------|------|------|--------|
| `ADM-07-AC-reward-list-filter` | 流水筛选 | 正常 | P0 |
| `ADM-07-AC-reward-list-export` | 导出 | 正常 | P1 |
