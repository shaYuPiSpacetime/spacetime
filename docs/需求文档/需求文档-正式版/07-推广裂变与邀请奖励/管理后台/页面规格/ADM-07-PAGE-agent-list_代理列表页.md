## 1. 页面定位

- **目标用户**：渠道运营、超管（运营/财务可查看统计）
- **核心任务**：录入与管理校园代理，承载代理统计概览入口
- **页面类型**：列表页

## 2. 布局

### 2.1 整体布局

```
┌──────────────────────────────────────────────┐
│ 搜索栏：代理编号、代理名称、学校、合作状态         │
├──────────────────────────────────────────────┤
│ 工具栏：新增代理 | 导出                          │
├──────────────────────────────────────────────┤
│ 列表（含累计统计 + 状态 + 操作）                 │
```

### 2.2 区块说明

| 区块 | 位置 | 内容 | 是否可折叠 | 是否记住展开状态 |
|------|------|------|-----------|----------------|
| 搜索栏 | 顶部 | 名称/学校/合作状态 | 是 | 否 |
| 列表 | 主体 | 代理信息 + 累计统计 + 应/已/待结算奖金 | 否 | 否 |

### 2.3 弹层

| 弹层 | 触发方式 | 大小 | 内容 | 关闭方式 |
|------|----------|------|------|----------|
| 新增/编辑代理 | 点击新增/编辑 | 中 | 代理信息表单 | 取消/保存 |

## 7. 状态与异常

| 状态类型 | 触发场景 | 页面表现 | 用户可做的操作 | 引用 |
|----------|----------|----------|---------------|------|
| 加载态 | 首次/查询 | 骨架屏 | 无 | — |
| 空态（无数据） | 0 条 | 插画 +「暂无代理，去新增」 | 新增 | — |
| 错误态 | 请求失败 | toast + 重试 | 重试 | — |
| 无权限态 | 运营/财务 | 新增/编辑/状态操作置灰，仅可查看 | 查看 | ADM-07_端内定义.md 第 2.2 节 |
| 业务态-暂停推广 | 合作状态=暂停推广 | 「暂停推广」胶囊 | 恢复/终止 | `M07-SM-agent-coop` |
| 业务态-终止合作 | 合作状态=终止合作 | 「终止合作」胶囊 | 查看 | `M07-SM-agent-coop` |
| 降级态（二维码） | 新增时二维码生成失败 | 提示稍后在素材页补生成 | 去素材页 | `M07-SRV-wx-qrcode` |

## 9. 验收标准

```
AC-ID: ADM-07-AC-agent-add
Given 渠道运营角色（GLB-ROLE-channel-op）
When  点击新增代理，填写名称/联系人/电话/学校/奖金规则组并保存
Then  生成代理编号与专属二维码（M07-SRV-wx-qrcode），新增记审计日志，列表出现该代理

AC-ID: ADM-07-AC-agent-terminate
Given 一个合作状态=正常推广的代理
When  点击终止合作并确认
Then  合作状态→终止合作（M07-SM-agent-coop），停止计奖，记审计；新用户扫码仍可进小程序

AC-ID: ADM-07-AC-agent-no-permission
Given 财务角色（GLB-ROLE-finance）
When  进入代理列表
Then  可查看统计与奖金，新增/编辑/状态操作按钮置灰
```

### 验收标准清单

| AC ID | 场景 | 类型 | 优先级 |
|-------|------|------|--------|
| `ADM-07-AC-agent-add` | 新增代理 | 正常 | P0 |
| `ADM-07-AC-agent-terminate` | 终止合作 | 正常 | P0 |
| `ADM-07-AC-agent-no-permission` | 无权限 | 异常 | P0 |
