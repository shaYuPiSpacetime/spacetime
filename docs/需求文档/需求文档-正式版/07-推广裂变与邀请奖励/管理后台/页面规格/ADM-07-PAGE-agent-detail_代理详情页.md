# 页面规格 - ADM-07-PAGE-agent-detail 代理详情页

> 涉及枚举/状态/权限/错误码/文案，一律引用全局定义、模块公共定义或模块端内定义。

| 版本 | 日期 | 修改人 | 变更摘要 |
|------|------|--------|----------|

- **页面 ID**：`ADM-07-PAGE-agent-detail`
- **所属模块 PRD**：`../模块PRD文档/模块PRD_ADM-07_推广管理.md`
- **页面路由**：`/admin/promotion/agent/:id`（实现可使用内部 ID；页面标题与展示字段优先展示代理业务标识/代理名称）
- **入口来源**：代理列表「查看详情」
- **对应移动端**：—

---

## 1. 页面定位

- **目标用户**：渠道运营、财务、运营、超管
- **核心任务**：查看代理推广明细、奖金明细、结算记录
- **页面类型**：详情页（多 Tab）

---

## 2. 布局

### 2.1 整体布局

```
┌──────────────────────────────────────────────┐
│ 顶部：代理信息卡 + 合作状态 + 累计统计概览       │
├──────────────────────────────────────────────┤
│ Tab: 推广明细 | 奖金明细 | 结算记录              │
├──────────────────────────────────────────────┤
│ 对应 Tab 列表                                   │
└──────────────────────────────────────────────┘
```

### 2.2 区块说明

| 区块 | 位置 | 内容 | 是否可折叠 | 是否记住展开状态 |
|------|------|------|-----------|----------------|
| 代理信息概览 | 顶部 | 名称/学校/合作状态/累计统计 | 否 | 否 |
| 推广明细 Tab | Tab1 | 被推广用户明细 | 否 | 是 |
| 奖金明细 Tab | Tab2 | 奖金事件明细 | 否 | 是 |
| 结算记录 Tab | Tab3 | 结算单记录 | 否 | 是 |

---

## 3. 标题与概览

| 项 | 内容 |
|----|------|
| 页面标题 | 代理详情 |
| 主标题 | 代理名称 + 代理编号（`AGT-yyyymmdd-xxxx`） |
| 副标题 | 学校/机构、负责人、合作状态中文名称、最近更新时间 |
| 状态标识 | 合作状态仅展示中文名称：正常推广/暂停推广/终止合作 |
| 顶部统计 | 累计注册人数、累计成功邀请人数、累计奖金、待结算奖金、已发放奖金 |

## 7. 状态与异常

| 状态类型 | 触发场景 | 页面表现 | 用户可做的操作 | 引用 |
|----------|----------|----------|---------------|------|
| 加载态 | 首次/切 Tab | 骨架屏 | 无 | — |
| 空态（无数据） | 某 Tab 0 条 | 插画 +「暂无数据」 | 无 | — |
| 错误态 | 请求失败 | toast + 重试 | 重试 | — |
| 数据为空（代理不存在） | 代理编号无效 | 空态 + 返回入口 | 返回列表 | — |
| 无权限态 | 页面权限受限 | 推广明细按权限展示/导出按权限置灰 | 查看 | ADM-07_端内定义.md 第 2.2 节 |

## 8. 查询与列表

- **默认排序**：各 Tab 按时间倒序
- **分页**：默认 20 条/页
- **批量选择**：不支持

## 9. 验收标准

```
AC-ID: ADM-07-AC-agent-detail-stat
Given 一个有推广数据的代理
When  进入详情页推广明细 Tab
Then  展示被推广用户 UUID/姓名/手机号（敏感字段按权限展示）、推广关系状态；顶部累计成功邀请人数按 M07-RULE-invite-success 配置节点统计

AC-ID: ADM-07-AC-agent-detail-bonus
Given 代理有多条奖金记录
When  进入奖金明细 Tab
Then  展示奖金明细订单号、奖金事件、金额、状态中文名称（M07-ENUM-agent-bonus-status）
```

### 验收标准清单

| AC ID | 场景 | 类型 | 优先级 |
|-------|------|------|--------|
| `ADM-07-AC-agent-detail-stat` | 推广明细 | 正常 | P0 |
| `ADM-07-AC-agent-detail-bonus` | 奖金明细 | 正常 | P0 |
