## 1. 页面定位

- **目标用户**：渠道运营、财务、超管
- **核心任务**：查询与管理代理奖金结算台账（首版不做自动打款，线下发放）
- **页面类型**：列表页（状态流转）

---

## 2. 布局

### 2.1 整体布局

```
┌──────────────────────────────────────────────┐
│ 搜索栏：代理编号/名称、结算周期、结算状态          │
├──────────────────────────────────────────────┤
│ 工具栏：导出                                    │
├──────────────────────────────────────────────┤
│ 结算单列表（含金额 + 状态 + 操作）               │
└──────────────────────────────────────────────┘
```

### 2.2 区块说明

| 区块 | 位置 | 内容 | 是否可折叠 | 是否记住展开状态 |
|------|------|------|-----------|----------------|
| 搜索栏 | 顶部 | 代理编号/名称/周期/状态 | 是 | 否 |
| 结算单列表 | 主体 | 结算单台账 | 否 | 否 |

### 2.3 弹层

本页不提供生成结算单弹层。结算单由 PRD-07 系统结算任务按结算周期生成（`M07-RULE-agent-settlement-generate`，无页面入口），本页只负责查询、确认、标记已发放与导出。

## 5. 操作表

### 5.1 行级操作

| 操作 ID | 操作名 | 触发条件 | 前置权限 | 二次确认 | 成功态 | 失败态 | 影响 |
|---------|--------|----------|----------|----------|--------|--------|------|
| `...-ACT-confirm` | 标记已确认 | 结算状态=待结算 | 渠道运营/财务/超管 | 是 | 结算状态→已确认，记审计 | `M07-ERR-*` | 刷新 |
| `...-ACT-mark-paid` | 标记已发放 | 结算状态=已确认 | 财务/超管 | 是「确认线下已发放？」 | 结算状态→已发放，记审计 | `M07-ERR-*` | 刷新 |
| `...-ACT-export-detail` | 导出结算明细 | — | 财务/超管 | 否 | 异步生成 | 重试 |

### 5.3 页面级操作

| 操作 ID | 操作名 | 位置 | 触发条件 | 前置权限 | 二次确认 | 成功态 | 失败态 |

## 7. 状态与异常

| 状态类型 | 触发场景 | 页面表现 | 用户可做的操作 | 引用 |
|----------|----------|----------|---------------|------|
| 加载态 | 首次/查询 | 骨架屏 | 无 | — |
| 空态（无数据） | 0 条 | 插画 +「暂无结算单」 | 调整筛选条件 | — |
| 错误态 | 请求失败 | toast + 重试 | 重试 | — |
| 无权限态 | 运营/风控 | 页面不可见或操作置灰 | 无 | ADM-07_端内定义.md 第 2.2 节 |
| 业务态-待结算 | 结算状态=待结算 | 「待结算」胶囊 | 确认 | `M07-SM-agent-bonus` |
| 业务态-已确认 | 结算状态=已确认 | 「已确认」胶囊 | 标记已发放/导出 | `M07-ENUM-settlement-status` |
| 业务态-已发放 | 结算状态=已发放 | 「已发放」胶囊 | 查看/导出 | — |

## 9. 验收标准

```
AC-ID: ADM-07-AC-settle-list-show
Given PRD-07 系统结算任务已生成代理结算单
When  财务进入代理结算管理页
Then  展示结算单号、代理编号/名称、结算周期、应结算金额、结算状态中文名称

AC-ID: ADM-07-AC-settle-mark-paid
Given 一张结算状态=已确认的结算单，财务角色（GLB-ROLE-finance）
When  点击标记已发放并确认
Then  结算状态→已发放（M07-SM-agent-bonus/M07-ENUM-settlement-status），记审计日志

AC-ID: ADM-07-AC-settle-no-permission
Given 渠道运营角色
When  尝试标记已发放
Then  无权限，按钮置灰（仅财务/超管可标记已发放）
```

### 验收标准清单

| AC ID | 场景 | 类型 | 优先级 |
|-------|------|------|--------|
| `ADM-07-AC-settle-list-show` | 结算单查询 | 正常 | P0 |
| `ADM-07-AC-settle-mark-paid` | 标记已发放 | 正常 | P0 |
| `ADM-07-AC-settle-no-permission` | 无权限 | 异常 | P0 |
