## 1. 页面定位

- **目标用户**：运营、风控、超管
- **核心任务**：查看单条邀请关系全链路（时间轴、奖励、风控、操作日志）
- **页面类型**：详情页

---

## 2. 布局

### 2.1 整体布局

```
┌──────────────────────────────────────────────┐
│ 顶部：关系状态胶囊 + 操作（解冻/判无效）        │
├──────────────────┬───────────────────────────┤
│ 邀请人信息        │ 被邀请人信息（按权限展示）  │
├──────────────────┴───────────────────────────┤
│ 关键时间轴：点击/注册/登录/资料完善/认证完成/付费 │
├──────────────────────────────────────────────┤
│ 奖励触发记录                                    │
├──────────────────────────────────────────────┤
│ 风控命中记录                                    │
├──────────────────────────────────────────────┤
│ 操作日志                                        │
└──────────────────────────────────────────────┘
```

### 2.2 区块说明

| 区块 | 位置 | 内容 | 是否可折叠 | 是否记住展开状态 |
|------|------|------|-----------|----------------|
| 关系状态与操作 | 顶部 | 状态胶囊 + 解冻/判无效按钮 | 否 | 否 |
| 双方信息 | 上部 | 邀请人/被邀请人信息（手机号等敏感字段按页面权限展示） | 否 | 否 |
| 关键时间轴 | 中部 | 首次点击/注册/成功登录/资料完善/认证完成/首次会员/首次充值时间 | 否 | 否 |
| 奖励触发记录 | 中部 | 各奖励事件记录及状态 | 是 | 否 |
| 风控命中记录 | 下部 | 命中原因与时间 | 是 | 否 |
| 操作日志 | 底部 | 人工处理记录 | 是 | 否 |

### 5.3 页面级操作

## 7. 状态与异常

| 状态类型 | 触发场景 | 页面表现 | 用户可做的操作 | 引用 |
|----------|----------|----------|---------------|------|
| 加载态 | 首次加载 | 骨架屏 | 无 | — |
| 错误态 | 请求失败 | toast + 重试 | 重试 | — |
| 数据为空（对象不存在） | 邀请关系编号无效 | 空态 + 返回入口 | 返回列表 | — |
| 无权限态 | 无权限角色 | 操作置灰，敏感信息展示按权限控制 | 无 | ADM-07_端内定义.md 第 2.2 节 |

| 业务态-成功统计节点达成 | 达到 `M07-RULE-invite-success` 配置节点 | 时间轴对应节点高亮 + 成功标记 | 查看 | `M07-RULE-invite-success` |

## 9. 验收标准

```
AC-ID: ADM-07-AC-relation-detail-show
Given 一条状态=已完成认证的邀请关系
When  进入详情页
Then  展示邀请来源、双方信息（敏感字段按权限展示）、完整时间轴、奖励触发记录、风控记录、操作日志

AC-ID: ADM-07-AC-relation-detail-unfreeze
Given 一条状态=风险冻结中的关系，风控角色
When  在详情页点击解除冻结并确认
Then  关联奖励→已到账，操作日志新增一条记录
```

### 验收标准清单

| AC ID | 场景 | 类型 | 优先级 |
|-------|------|------|--------|
| `ADM-07-AC-relation-detail-show` | 详情展示 | 正常 | P0 |
| `ADM-07-AC-relation-detail-unfreeze` | 详情解冻 | 正常 | P0 |
