## 2. 角色与权限矩阵

### 2.1 角色引用

| 角色 code | 显示名 | 本模块中承担的职责 |
|-----------|--------|-------------------|
| `GLB-ROLE-operation` | 运营 | 查看与配置普通裂变规则、查看邀请数据、配置文案 |
| `GLB-ROLE-channel-op` | 渠道运营 | 新增代理、管理二维码、查看代理统计、确认结算台账 |
| `GLB-ROLE-finance` | 财务/结算 | 查看代理奖金台账、标记已发放、导出结算单 |
| `GLB-ROLE-risk` | 风控 | 查看冻结记录、处理异常奖励 |
| `GLB-ROLE-super-admin` | 超级管理员 | 修改全部规则、处理敏感状态、查看完整审计日志 |

### 2.2 权限矩阵（ADM-07 涉及操作）

> 权限最终以后台菜单管理/页面操作权限配置为准；下表为首版建议映射。可看到对应页面的人可执行该页面授权操作；冻结处理与结算确认由有页面权限的人处理，SLA 人工判断。
> 单元格：允许 / 禁止。

| 操作 / 能力 | 运营 | 渠道运营 | 财务 | 风控 | 超管 |
|-------------|------|----------|------|------|------|
| 配置普通裂变奖励规则 | 允许 | 禁止 | 禁止 | 禁止 | 允许 |
| 配置代理奖励规则 | 禁止 | 允许 | 禁止 | 禁止 | 允许 |
| 配置风控参数 | 禁止 | 禁止 | 禁止 | 允许 | 允许 |
| 查看普通邀请关系/奖励 | 允许 | 允许 | 允许 | 允许 | 允许 |
| 人工解除冻结/判无效 | 禁止 | 禁止 | 禁止 | 允许 | 允许 |
| 新增/编辑/停用代理 | 禁止 | 允许 | 禁止 | 禁止 | 允许 |
| 管理代理二维码与素材 | 禁止 | 允许 | 禁止 | 禁止 | 允许 |
| 查看代理统计 | 允许 | 允许 | 允许 | 禁止 | 允许 |
| 查看/确认结算台账 | 禁止 | 允许 | 允许 | 禁止 | 允许 |
| 标记已发放/导出结算 | 禁止 | 禁止 | 允许 | 禁止 | 允许 |
| 配置推广文案 | 允许 | 允许 | 禁止 | 禁止 | 允许 |

> **数据权限：** 渠道运营默认仅可见本人/本组负责代理数据；财务可见全部奖金结算数据；运营可见全部邀请数据。推广相关导出包含手机号/收款信息且不脱敏，导出动作必须审计。
> **结算单生成边界：** ADM-07 代理结算管理页不提供手动生成结算单入口；结算单由 PRD-07 系统结算任务按周期生成（`M07-RULE-agent-settlement-generate`，无页面入口），ADM-07 负责台账查询、确认、发放标记与导出。

### 2.3 敏感操作审计要求

| 操作 | 是否二次确认 | 是否记录审计日志 | 审计日志保留 |
|------|-------------|-----------------|-------------|
| 修改普通邀请奖励规则 | 是 | 是 | >= 1 年 |
| 修改代理奖励规则 | 是 | 是 | >= 1 年 |
| 修改风控参数 | 是 | 是 | >= 1 年 |
| 手工解除冻结 | 是 | 是 | >= 1 年 |
| 手工判无效 | 是 | 是 | >= 1 年 |
| 新增/编辑/停用代理 | 是 | 是 | >= 1 年 |
| 变更结算状态 | 是 | 是 | >= 1 年 |
| 标记奖金已发放 | 是 | 是 | >= 1 年 |
| 导出推广/结算明细 | 是 | 是 | >= 1 年 |

### 2.4 后台展示标识规则

> ADM-07 后台列表、详情、搜索条件、导出字段等用户可见展示层尽量不直接展示数据库自增主键 ID。接口、路由参数、内部定位与前后端联调仍可使用内部 ID；页面需要定位记录时优先展示业务标识。

| 对象 | 后台展示标识 | 说明 |
|------|--------------|------|
| 用户 | 用户 UUID + 昵称/手机号（按权限脱敏） | 不展示内部主键 ID |
| 代理 | 代理编号 `AGT-yyyymmdd-xxxx` + 代理名称 | 不展示内部主键 ID |
| 邀请关系 | 邀请关系编号 `REL-yyyymmdd-xxxx` | 不展示内部主键 ID |
| 普通邀请奖励流水 | 奖励流水号 `IRW-yyyymmdd-xxxx` | 不展示内部主键 ID |
| 代理奖金记录 | 奖金流水号 `ABN-yyyymmdd-xxxx` | 不展示内部主键 ID |
| 结算单 | 结算单号 `SET-yyyymm-xxxx` | 不展示内部主键 ID |
| 二维码 | 二维码编号 `QR-yyyymmdd-xxxx` | 不展示内部主键 ID |

> 详情页可见标题与面包屑优先展示业务标识；路由参数允许使用内部 `:id`，也可按页面实现使用业务标识。

### 2.5 字段展示格式规则

| 字段类型 | 后台展示要求 | 说明 |
|----------|--------------|------|
| 枚举 / 状态 | 只展示中文名称 | 列表、详情、筛选项、状态胶囊、操作条件、验收标准、导出均展示中文；code 仅作为接口/字典内部值或文档引用 ID |
| 时间 | `yyyy-MM-dd HH:mm:ss` | 所有 datetime 类型字段在列表、详情、导出中统一展示到秒 |

## 3. UI 空态 / 错误态 / 降级态

| 场景 | 表现 | 默认文案 |
|------|------|----------|
| 列表无数据 | 居中插画 + 文案 | 暂无数据 |
| 搜索无结果 | 居中插画 + 清除筛选建议 | 未找到相关记录 |
| 请求失败（网络/服务端） | toast + 重试按钮 | 加载失败，请重试 |
| 无权限 | 菜单/按钮不展示或置灰 | 无操作权限 |
| 二维码生成失败（降级） | 占位 + 重试 | 二维码生成失败，请重试 |
| 导出生成中 | loading + 异步生成提示 | 导出生成中，请稍后在下载中心查看 |
