## 2. 登录态与权限

移动端无后台角色概念，本节仅区分未登录、已登录普通用户。

| 用户状态 | 可见入口 | 可执行操作 | 不满足时处理 |
|----------|----------|------------|--------------|
| 未登录 | 可见推荐入口 | 不可生成二维码/分享/查看记录 | 点击后引导登录 |
| 已登录普通用户 | 可见推荐入口 | 生成二维码、分享、查看进度与记录 | — |

## 3. UI 空态 / 错误态 / 降级态

| 场景 | 表现 | 默认文案 |
|------|------|----------|
| 邀请记录无数据 | 居中插画 + 文案 + 分享引导 | 还没有邀请记录，快邀请同学一起玩 |
| 请求失败（网络） | toast + 保留上次数据 | 网络开小差了，请稍后重试 |
| 请求失败（服务端） | toast + 重试按钮 | 加载失败，点击重试 |
| 二维码生成失败（降级） | 二维码区占位 + 重试按钮，不阻塞其余区块 | 二维码生成失败，请稍后重试 |
| 微信分享不可用（降级） | 隐藏分享按钮，引导保存二维码图片转发 | 请保存二维码分享给好友 |
| 奖励冻结提示 | 记录项标记冻结态 + 提示 | 奖励正在核验中，请稍后查看 |
| 奖励无效提示 | 记录项标记无效态 + 原因 | 该邀请未满足活动规则，未计入奖励 |
