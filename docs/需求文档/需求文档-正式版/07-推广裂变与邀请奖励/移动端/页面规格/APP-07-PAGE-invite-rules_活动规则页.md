## 1. 页面定位

- **目标用户**：邀请人
- **核心任务**：解释成功邀请口径与奖励规则
- **页面类型**：内容展示页（静态文案为主）

---

## 2. 布局

### 2.1 整体布局

```
┌─────────────────────────────────────────┐
│ 成功邀请定义                              │
├─────────────────────────────────────────┤
│ 阶梯奖励示意（后台配置档位与币数）          │
├─────────────────────────────────────────┤
│ 单日奖励上限说明                          │
├─────────────────────────────────────────┤
│ 风控与无效规则                            │
├─────────────────────────────────────────┤
│ 联系客服入口                              │
└─────────────────────────────────────────┘
```

### 2.2 区块说明

| 区块 | 位置 | 内容 | 是否可折叠 | 是否记住展开状态 |
|------|------|------|-----------|----------------|
| 成功邀请定义 | 顶部 | 新用户 + 通过邀请链接进入 + 达到后台配置的成功统计节点（`M07-RULE-invite-success`），文案 `M07-TXT-invite-success-def` | 否 | 否 |
| 阶梯奖励示意 | 中部 | 展示后台配置的阶梯档位与币数（`M07-RULE-invite-ladder`），文案 `M07-TXT-invite-ladder-desc` | 否 | 否 |
| 日奖励上限 | 中部 | 单日上限说明（`M07-CFG-invite-daily-cap`，无默认值，未配置则不展示具体数字） | 否 | 否 |
| 风控与无效规则 | 下部 | 作弊/刷量/自邀不计奖、平台有权冻结复核（`M07-RULE-invite-antifraud`/`M07-RULE-invite-invalid`） | 否 | 否 |
| 联系客服 | 底部 | 客服入口 | 否 | 否 |

## 4. 字段表

### 4.1 展示字段

| 字段 ID | 显示名 | 类型 | 必填 | 取值范围 | 校验规则 | 默认值 | 可编辑 | 敏感级别 | 数据来源 |
|---------|--------|------|------|----------|----------|--------|--------|----------|----------|
| `APP-07-PAGE-invite-rules-FIELD-content` | 规则文案 | string | 是 | `M07-TXT-invite-rules-page` 等文案 key | 移动端只渲染后台返回内容，不写死文案 | 内置兜底文案（接口失败时） | 否（运营中心可配） | 普通 | `/api/promotion/invite/rules` |

> 规则页文案全部后台可配，引用 `M07-TXT-invite-rules-page` 等文案 key，移动端只渲染不写死。文案配置页面为 `ADM-GLB-PAGE-copy-message-center`（运营中心 → 文案与消息中心 → 推广裂变分组），不归属 PRD-06 认证与安全设置页。规则数字、成功统计节点、阶梯与日上限来自 `ADM-07-PAGE-promo-rule-config`。

## 9. 验收标准

```
AC-ID: APP-07-AC-rules-show
Given 已登录用户
When  进入活动规则页
Then  展示成功邀请定义（后台配置统计节点）、后台配置的阶梯示意、日上限、风控无效规则、客服入口，埋点 invite_rules_show 上报

AC-ID: APP-07-AC-rules-fallback
Given 后台规则文案接口失败
When  进入活动规则页
Then  展示内置兜底文案，不白屏
```

### 验收标准清单

| AC ID | 场景 | 类型 | 优先级 |
|-------|------|------|--------|
| `APP-07-AC-rules-show` | 规则展示 | 正常 | P1 |
| `APP-07-AC-rules-fallback` | 文案降级 | 异常 | P1 |
