# 移动端-资料编辑扩展

## 关键还原点

- 编辑资料总页展示资料完整度、头像认证状态、开放文本状态、相册数量和资料背景图缺失。
- 基础资料编辑页覆盖昵称、现居地、职业、公司等可编辑字段，并展示认证后锁定字段。
- 扩展资料编辑页覆盖关于我、兴趣标签、相册、资料背景图、语音介绍和开放文本审核联动。
- 补齐 Lanhu 缺口中资料编辑相关页面，避免只有认证入口而缺少用户后续维护资料的路径。
