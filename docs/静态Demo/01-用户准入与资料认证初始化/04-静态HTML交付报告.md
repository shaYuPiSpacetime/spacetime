## 4. 关键业务覆盖

- 登录授权：微信登录、手机号短信降级、用户协议和隐私政策、第三方不可用。
- 轻量资料：性别、出生日期、身份、最高学历、现居地、年龄拦截、定位失败。
- 资料认证：头像认证、实名认证、学历认证、自我介绍、语音介绍、单身承诺、三重认证。
- 核心准入：实名认证通过、头像认证通过、学历认证通过、账号正常；缺失时统一拦截。
- 资料编辑：编辑资料总页、基础资料编辑、扩展资料编辑、资料完整度。
- App 用户管理：查询条件、头部统计、表格列表、分页、画像详情抽屉、编辑备注、冻结账号、批量导入、导出确认。
- 准入配置：准入门槛、字段配置、资料完整度、上传限制、审核 SLA、文案配置、安全策略。
- 审核队列：头像、实名、学历、资料照片、开放文本五类审核，支持通过、驳回、人工复核。
