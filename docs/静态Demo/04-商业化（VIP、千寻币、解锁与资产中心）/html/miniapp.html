会员协议与连续订阅服务协议

会员套餐购买后立即生效；连续订阅套餐将通过微信小程序真实自动续费扣款。套餐价格和标签以后台配置快照为准。

支付暂不可用

微信支付服务不可用，支付按钮置灰或提示稍后重试，不创建成功资产。

取消续费确认

取消续费不会影响已支付会员有效期。确认后打开微信连续订阅管理入口或展示平台内取消步骤。
