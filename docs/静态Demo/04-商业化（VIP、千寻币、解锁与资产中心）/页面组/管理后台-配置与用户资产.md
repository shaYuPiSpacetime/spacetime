# 页面组：管理后台-配置与用户资产

## 1. 页面组范围

| 页面 ID | 页面名称 | 来源文件 |
|---------|----------|----------|
| `ADM-04-PAGE-commerce-config` | 商业化配置页 | `商业化配置页.md` |
| `ADM-04-PAGE-user-asset-detail` | 用户商业化详情内容区 | `用户商业化详情页.md`，由用户管理详情弹窗/详情页承载，本 Demo 不单独绘制 |

## 2. 页面组元素清单

| 页面 | 必须还原的元素 | 必须还原的状态 |
|------|----------------|----------------|
| 商业化配置 | 7 个 Tab、9 项会员权益、会员套餐、连续订阅套餐、千寻币套餐、6 个场景单价、保留期、社交与订单参数、曝光包预留、保存确认、日志抽屉 | 会员权益名称/类型/说明只读；免费悄悄话、额外浏览、每日心动机会配置次数，曝光配置分数；套餐弹窗字段分离、连续订阅配置缺失、保存失败 |
| 用户商业化详情 | 不在本 Demo 单独绘制 | 已由用户管理详情弹窗/详情页承载 |

## 3. HTML 片段映射

| 页面 | HTML 锚点 | 关键组件 |
|------|-----------|----------|
| 商业化配置 | `admin.html#ADM-04-PAGE-commerce-config` | `commerce-config-tabs`、`config-panel`、`configSaveModal`、`configLogDrawer` |
| 用户商业化详情 | 不提供独立 `admin.html` 锚点 | 外部用户管理详情弹窗承载 |

## 4. 评分预期

| 维度 | 预期 |
|------|------|
| 页面覆盖 | 2/2 全覆盖 |
| 字段控件 | 配置项可编辑/只读均区分 |
| 列表详情 | 用户详情 4 个 Tab 和抽屉可访问 |
| 操作反馈 | 保存、日志、人工修正均有确认/反馈 |
