# 页面组：管理后台-财务查询与对账

## 1. 页面组范围

| 页面 ID | 页面名称 | 来源文件 |
|---------|----------|----------|

## 2. 页面组元素清单

| 页面 | 必须还原的元素 | 必须还原的状态 |
|------|----------------|----------------|
| 订单管理 | 头部统计、筛选、订单表格、分页、订单详情抽屉、导出确认 | 待支付、成功、退款中、已退款、渠道延迟、空态 |
| 资产流水 | 头部统计、筛选、流水表格、分页、流水详情抽屉、关联订单、导出确认 | 充值/奖励收入、消费支出、PRD-07 未上线降级、空态 |
| 退款记录 | 头部统计、筛选、退款表格、分页、退款详情抽屉、导出确认 | 退款中、已退款、线下处理、渠道延迟、空态；不提供改状态 |
| 轻量对账 | 日期筛选、订单类型筛选、汇总卡、日维度表格、导出确认、更新时间 | 正常、无数据、统计延迟 |
