# 页面组：移动端-流水订单与付费弹窗

## 1. 页面组范围

| 页面 ID | 页面名称 | 来源文件 |
|---------|----------|----------|

## 2. 页面组元素清单

| 页面 | 必须还原的元素 | 必须还原的状态 |
|------|----------------|----------------|
| 千寻币流水 | 余额摘要、流水类型 Tab、流水卡片、变动数量、业务场景、发生时间、加载更多、充值入口 | 收入、支出、空态、加载失败 |
| 会员订单 | 订单号、套餐名、套餐类型、金额、状态、支付时间、到期时间、退款说明、加载更多 | 支付成功、退款中、已退款、空态 |
| 支付结果 | 订单号、订单类型、状态、结果标题、获得资产、来源页、返回来源、查看记录、重新支付、刷新 | 成功、失败、取消/关闭、资产刷新中 |
| 付费弹窗 | 会员引导、扣币确认、余额不足快捷充值、所需币数、当前余额、保留期说明、核心准入提示 | 余额充足、余额不足、核心准入受限、支付不可用 |
