## 2. 页面组元素清单

| 页面 | 必须还原的元素 | 必须还原的状态 |
|------|----------------|----------------|
| 会员中心 | 会员状态、到期时间、核心准入提示、权益列表、普通/连续订阅套餐、套餐标签、协议勾选、底部支付按钮、订阅管理入口 | 未开通、生效中、已过期、无套餐、支付不可用、连续订阅不可用 |
| 千寻币充值 | 当前余额、来源场景提示、套餐卡、支付金额、到账币数、购买说明、底部充值按钮 | 余额不足来源、无套餐、支付不可用 |
| 资产中心 | 会员卡、余额卡、免费悄悄话、千寻币流水入口、会员订单入口、连续订阅入口、刷新资产 | 新用户空记录、资产刷新失败、订阅服务不可用 |
| 连续订阅管理 | 订阅状态、套餐名、续费金额、周期、下次续费时间、协议说明、取消续费指引、订单入口 | 生效中、已取消、未开通、暂不可用 |

## 4. 评分预期

| 维度 | 预期 |
|------|------|
| 页面覆盖 | 4/4 全覆盖 |
| 字段控件 | 套餐卡、协议复选框、只读状态、入口列表均实现 |
| 操作反馈 | 支付、刷新、取消续费都有 toast/确认 |
| 状态异常 | 覆盖支付不可用、订阅不可用、资产刷新失败 |
