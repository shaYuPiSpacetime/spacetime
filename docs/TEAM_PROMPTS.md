# Spacetime 团队提示词速查

CLAUDE.md 已自动加载项目规范，以下提示词直接说就行，不需要教 AI 怎么干活。

---

## 一句话开发（最常用）

```
遵循项目规范，开发以下需求：【描述】
```

---

## 按阶段使用

觉得当前处于哪个阶段，就说对应的话。AI 会自动调 CLAUDE.md 里对应的 skill。

| 你现在的状态 | 说什么 |
|-------------|-------|
| 需求不清晰，不知道怎么做 | `这个需求我还不太清楚：【描述】，帮我梳理一下` |
| 需求清晰了，需要落地方案 | `为以下需求出个实施计划：【描述】` |
| 计划有了，开始写代码 | `执行计划`，或 `遵循项目规范，开发 XXX` |
| 先写测试再写代码 | `用 TDD 开发：【描述】` |
| 开发完了，检查一下 | `检查一下还有没有问题` |
| 提交前 Code Review | `帮我 review 一下当前改动` |
| 收到 review 反馈要修改 | `review 的意见：【内容】，帮我改` |
| 遇到 bug 了 | `帮我排查这个 bug：【描述】` |
| 联调测试前端页面 | `测试一下【页面路径】，检查功能正不正常` |
| 准备上线 | `准备上线，检查一下部署环境` |
| 提交代码 | `提交代码` |
