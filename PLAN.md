# 蓝湖登录资料四页 1:1 交互还原方案
蓝湖链接：https://lanhuapp.com/link/#/invite?sid=lXW7rf47
## Summary
- 目标页面：`miniapp/src/pages/login/gender.tsx`、`age.tsx`、`education.tsx`、`address.tsx`。
- 按蓝湖稿还原「性别、年龄、学历、地址」四页：无导航标题、共用一致背景、内容区标题/说明/卡片/按钮位置按 750×1624 稿件 rpx 坐标实现。
- 保留真实交互：性别/学历可点选，年龄使用原生 `PickerView`，地址点击选择城市或获取定位。

## Key Changes
- 新增登录资料页共用壳组件，统一处理：
  - 全屏一致背景层，基于蓝湖共同底色生成/使用 `miniapp/src/assets/login/` 下的背景图。
  - 无标题自定义导航，只显示左上返回箭头，不再传 `CustomNavBar title`。
  - 页面主标题固定为「请选择」，说明文案按蓝湖四张稿逐字配置。
  - 底部圆形下一步按钮使用蓝湖尺寸、颜色和白色箭头资源。
- 性别页：
  - 默认选中「我是女生」，下一步按钮默认高亮。
  - 两张性别卡片按蓝湖位置、圆角、边框、渐变图标和文字样式重排。
- 年龄页：
  - 使用原生 `PickerView`，但去掉现有白色卡片外观，改成蓝湖五行三列视觉。
  - 默认值为 `1997年 / 10月 / 1日`，选中行使用浅蓝背景、蓝色描边、24rpx 圆角。
- 学历页：
  - 默认高亮「硕士」，四个选项为「博士、硕士、本科、大专」。
  - 删除额外样式偏差，按蓝湖 700rpx 宽卡片、24rpx 圆角、选中蓝色描边实现。
- 地址页：
  - 删除现有城市快捷标签，默认只显示一行「选择城市 / 获取定位」。
  - 点击选择城市用现有城市列表弹出选择；点击获取定位调用小程序定位/选点能力，失败时 toast 提示。

## Public Interfaces / Types
- 不改后端接口、不新增 API URL、不改登录流程对外路由。
- 可新增本地展示组件和少量 UI helper 类型，仅服务这四个登录资料页。
- 页面配置继续保持 `navigationStyle: 'custom'`。

## Test Plan
- 按用户要求，改动完成后不执行编译、构建、测试或 Taro build。
- 静态验收点：
  - 四页顶部没有「选择性别/选择年龄/选择学历/选择地区」导航标题。
  - 四页背景一致，且不再使用当前蓝色渐变背景。
  - 默认态符合蓝湖：性别选中女生、年龄为 1997/10/1、学历选中硕士、地址未选。
  - 地址页没有额外城市标签；年龄页没有白色 Picker 卡片。
  - 点击下一步仍按原流程跳转：性别 → 年龄 → 学历 → 地址 → 首页。

## Assumptions
- 蓝湖 750×1624 设计尺寸直接按 `rpx` 还原。
- 年龄页按你选择的「原生滚轮」实现，因此交互滚动体验优先于纯静态截图像素；默认可见态仍尽量贴近蓝湖。
- 学历/年龄页底部按钮默认按截图浅蓝展示，用户交互后进入可继续的高亮态。
